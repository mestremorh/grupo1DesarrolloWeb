Quienes Somos

Estuduantes de Especialización en Desarrollo del Software

Seguridad en el Desarrollo de Software

El propósito de este curso es dejar claro que el desarrollo seguro de software se basa en la ejecución de verificaciones de seguridad de manera continua durante el ciclo de vida del proyecto de software, incluso antes de escribir la primera línea de código.

Desarrollo de Aplicaciones Web

Al realizar el curso de Desarrollo de Aplicaciones web, obtendrá un conjunto de herramientas, criterios y conocimientos para identificar si el sitio web, en el cual está navegando, corresponde a un sitio oficial o se trata de una suplantación o phishing. Por otro lado, a nivel profesional, le proporcionará herramientas y técnicas para generar buenas prácticas en el desarrollo de aplicaciones web y generar políticas para el uso o restricción de las mismas.

Investigación aplicada

La investigación aplicada es un enfoque de investigación que tiene como objetivo abordar problemas prácticos y generar conocimientos y soluciones que puedan aplicarse directamente en el mundo real. A diferencia de la investigación pura o básica, que busca ampliar el conocimiento teórico sin una aplicación específica inmediata, la investigación aplicada tiene un enfoque más orientado hacia la acción y la resolución de problemas concretos.

Administración y Gestión de la Configuración del Software

La Administración y Gestión de la Configuración del Software (AGCS) es una disciplina dentro de la ingeniería de software que se ocupa de gestionar y controlar los elementos de configuración de un sistema de software durante todo su ciclo de vida.
La configuración del software se refiere a todos los componentes y elementos que conforman un sistema de software, incluyendo el código fuente, los archivos de configuración, la documentación, las bibliotecas, los scripts y cualquier otro artefacto relacionado. La AGCS tiene como objetivo principal garantizar que todos estos elementos estén organizados, versionados, controlados y disponibles de manera eficiente y precisa a lo largo del tiempo.

Plataformas de Desarrollo de Software

Las plataformas de desarrollo de software generalmente tienen como objetivo proporcionar a los desarrolladores una comprensión profunda de las diferentes plataformas y herramientas disponibles para el desarrollo de software. Algunos temas que podrían cubrirse incluyen:

• Una visión general de las diferentes plataformas y herramientas utilizadas en el desarrollo de software.

• Plataformas de desarrollo web: Exploración de las plataformas y frameworks más populares para el desarrollo de aplicaciones web.

• Plataformas de desarrollo móvil: Estudio de las plataformas para el desarrollo de aplicaciones móviles.

• Plataformas de desarrollo en la nube: Exploración de las plataformas y servicios en la nube más utilizados.

• Herramientas de desarrollo integrado (IDE): Introducción a los IDEs más populares utilizados en el desarrollo de software.

• Metodologías y prácticas de desarrollo: Discusión sobre las metodologías ágiles y enfoques de desarrollo modernos.

• Aspectos de seguridad en las plataformas de desarrollo: Consideración de las prácticas de seguridad en el desarrollo de software y cómo aplicar medidas de seguridad en las diferentes plataformas y entornos de desarrollo.

• Tendencias y novedades en plataformas de desarrollo: Exploración de las últimas tendencias, tecnologías y herramientas emergentes en el desarrollo de software, y cómo estas pueden influir en la elección de una plataforma de desarrollo.
